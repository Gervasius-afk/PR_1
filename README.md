# Задание: Мигание светодиодом по сигналу SOS (азбука Морзе)

## Требования
- Используйте встроенный светодиод на пине 13.
- Точка = 200 мс, тире = 600 мс.
- Пауза между символами — 200 мс, между буквами — 600 мс.
- После выполнения сигнала — пауза 2 секунды, затем повтор.

## Критерии оценки
- Корректная длительность импульсов (±20 мс)
- Читаемость кода (функции, комментарии)
- Отсутствие `delay()` в основной логике (использовать `millis()` — на «5»)

## Сдача
Загрузите файл `.ino` в этот репозиторий до дедлайна.
